# Data_Engineer

## Overview
This project demonstrates an end-to-end ETL pipeline using Python.

## Technologies
- Python
- Requests
- Pandas
- SQLite

## Workflow
API → Extract → Transform → Load → Analyze → Report

## Data Source
https://dummyjson.com/users

## Outputs
- gender_distribution.csv
- average_age_by_gender.csv
- top_10_oldest_users.csv
- users_by_state.csv

## Skills Demonstrated
- API Integration
- JSON Processing
- Data Cleaning
- SQL Analysis
- ETL Design
